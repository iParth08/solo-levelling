# Git commands for Use

## Logs

1. Oneline logs

```bash
git log --oneline
```

## Branch Timelines

1. All Branch

```bash
    git branch
```

2. New Branch

```bash
    git branch new-branch
```

3. Move to a Branch

```bash
    git checkout branch-name
```
